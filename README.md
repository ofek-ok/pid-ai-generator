# PID AI Generator - מחולל מסמכי ייזום פרויקטים

כלי ווב יוקרתי ומודרני המיועד למנהלי פרויקטים (PMs) ובעלי תפקידי PMO בארגונים, שנועד לייעל ולזרז את תהליך הכתיבה של מסמכי ייזום פרויקטים (PID - Project Initiation Documents). 

הכלי מאפשר להזין נתוני יסוד עבור **סעיפים 1 עד 5** באמצעות אשף (Wizard) אינטראקטיבי וידידותי, ולאחר מכן משתמש ב-Gemini API כדי לבצע הרחבה מקצועית, עיבוי ניסוחים ויצירה אוטומטית של **סעיפים 6 עד 12** (לוחות זמנים, בעלי עניין, קונספט פתרון, ניהול סיכונים, מטריצת RACI, מדדי הצלחה וטבלאות אישורים).

---

## ✨ תכונות עיקריות

*   **דף נחיתה שיווקי ומזמין**: דף בית מעוצב המציג את ערכי המוצר ומוביל ישירות לשימוש בכלי ללא צורך בהרשמה או התחברות.
*   **אשף מונחה שלבים (שאלון סעיפים 1-5)**: חלוקה נוחה לפרטי פרויקט, רקע ומצב קיים, מטרות עסקיות (כולל בונה משפט קיום ייחודי), תכולות (IN/OUT Scope) והערכת משאבים.
*   **מחולל AI מובנה (Gemini API)**: חיבור ישיר ורשמי למודלי Gemini 2.5 Flash / Pro המאובטחים של גוגל בצד הלקוח בלבד.
*   **מצב הדגמה (Demo Mode / Mock Mode)**: ניתן להתנסות ולבחון את האפליקציה באופן מלא ללא צורך במפתח API (האפליקציה תספק מסמך ייזום לדוגמה באיכות גבוהה על בסיס הנתונים שהזנת).
*   **עורך Markdown מפוצל**: מסך עבודה מתקדם המציג את קוד ה-Markdown של המסמך לעריכה חיה מצד שמאל, ותצוגה מקדימה מעוצבת לפי סטנדרט תאגידי מצד ימין.
*   **שמירה מקומית אוטומטית (Local Storage)**: מפתח ה-API והטיוטות שלך נשמרים בדפדפן המקומי בלבד. המידע העסקי שלך נשאר פרטי ומאובטח.
*   **דו-לשוני מלא (עברית / אנגלית)**: תמיכה מלאה בייצוא, כתיבה וממשק בשתי השפות כולל היפוך כיוון ממשק (RTL/LTR).
*   **ייצוא מקצועי**:
    *   **הורדה ל-Word**: ייצוא לקובץ `.doc` עצמאי עם סגנונות מובנים שנפתח ישירות ב-Microsoft Word תוך שמירה על טבלאות, צבעים ועיצוב.
    *   **הדפסה ל-PDF**: התאמה של סגנונות הדפסה (`@media print`) המייצרת קובץ PDF נקי, מעומד ומקצועי לחתימות ישירות מתוך תיבת הדו-שיח של הדפדפן.

## 🚀 הוראות הרצה מהירה

האפליקציה נבנתה כ-Single Page Application (SPA) טהורה עם HTML, CSS ו-JS ללא תלויות חיצוניות מורכבות, מה שהופך אותה לניידת ומהירה במיוחד.

### אפשרות א': הרצה ישירה בדפדפן (ללא התקנות)
פשוט לחץ לחיצה כפולה על הקובץ **`index.html`** והוא ייפתח בכל דפדפן מודרני (Chrome, Edge, Safari, Firefox).

### אפשרות ב': הרצה עם שרת פיתוח מקומי (מומלץ)
אם ברצונך להריץ אותה דרך שרת פיתוח קטן כדי לדמות סביבת ייצור:
```bash
# שימוש ב-npx (דרוש Node.js מותקן)
npx serve ./

# או שימוש ב-Python (מובנה ברוב מערכות ההפעלה)
python -m http.server 8000
```
לאחר מכן פתח את הדפדפן בכתובת: `http://localhost:8000` (או הכתובת שהוצגה בטרמינל).

## 🛠️ הגדרת מפתח ה-API של Gemini

על מנת שה-AI יוכל לייצר מסמכים אמיתיים עבור הפרויקטים שלך:
1.  לחץ על כפתור **"הגדרות API"** בפינה השמאלית העליונה בממשק של לוח הבקרה.
2.  הזן את מפתח ה-API של Gemini שלך (ניתן להנפיק מפתח בחינם תוך דקה בכתובת: [Google AI Studio](https://aistudio.google.com/)).
3.  בחר את המודל המבוקש (מומלץ להשתמש ב-**Gemini 2.5 Flash** שהוא מהיר ויעיל).
4.  לחץ על **"שמור הגדרות"**. המפתח יישמר מקומית בדפדפן ומעתה האפליקציה תבצע יצירת מסמכים חיה.
